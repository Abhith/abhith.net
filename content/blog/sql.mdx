Just execute the below query in the database to see the list of tables and number of records in each table.

```sql
 WITH CTE AS(
   SELECT [Col1]
      ,[Col2]
      ,..
      ,ROW_NUMBER() OVER(PARTITION BY [Col1] ORDER BY [Col1]) AS RN
   FROM [dbo].[Table]
)
DELETE FROM CTE WHERE RN > 1
```
